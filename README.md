# 🃏 Pad Game - 卡牌、寵物與技能進化的華麗戰鬥！

歡迎來到 **Pad Game**！這是一個以卡牌戰鬥為核心，結合寵物收集、技能進化和豐富視聽效果的遊戲項目。我們正在打造一個讓玩家沉浸其中、策略與養成並重的戰鬥體驗！

> ✨ **核心特色**：流暢的卡牌戰鬥、可愛的寵物夥伴、華麗的技能進化、統一的視覺風格，還有超讚的音樂與語音！


## 🎮 遊戲亮點

- **卡牌系統**：策略組合，打出你的最強連擊！
- **寵物與技能**：收集、培養、進化——你的寵物會越來越強！
- **裝備與覺醒**：武裝你的卡牌，解鎖隱藏力量！
- **狀態效果**：buff、debuff、各種有趣的效果，戰鬥變化多端！(實現了一半w
- **視聽盛宴**：精美的圖標、流暢的動畫、熱血的語音和背景音樂！(還沒實現wwwww)
- **直觀 UI**：一目了然的狀態顯示，點擊就能看到詳細效果說明。

## 🏗️ 專案架構

我們把程式碼整理得清清楚楚，方便大家一起開發：

```
📁 config/          # 設定檔（技能、效果、寵物數據）
📁 assets/          # 所有資源（圖標、音效、美術）
📁 src/
  ├── core/         # 核心戰鬥邏輯與傷害計算
  ├── ui/           # 選單、狀態顯示、頭像區域
  ├── assets/       # 資源管理（加載、圖標池）
  ├── audio/        # 語音、BGM、音效系統
  ├── vfx/          # 演出與特效
  ├── status/       # 狀態效果、覺醒、技能效果管理
  └── pets/         # 寵物、技能、進化數據結構
```

**設計理念**：數據驅動！技能、效果、圖標都透過設定檔管理，未來要調整或本地化都超簡單～
