# 轉珠遊戲 - PAD Clone

一個模仿龍族拼圖（Puzzle & Dragons）的益智轉珠遊戲

## 功能特色

### 分頁系統
- **背包**：顯示已拥有的寵物
- **圖鑑**：顯示所有寵物（含未获得）
- 底部 Tab 切換

### 背包系統
- 6x4 網格背包，共 24 個格子
- 拖曳移動：長按拖曳可移動寵物位置
- 點擊查看：點擊查看寵物詳細資訊

### 圖鑑系統
- 顯示所有寵物（目前 16 種）
- 已收集 / 總數 計數
- 未获得的寵物顯示為鎖定

### 寵物卡片系統
- 顯示名稱、種族、屬性
- 稀有度星級顯示
- 屬性顏色標示（火/水/木/光/暗）
- 卡片中央顯示屬性邊框

### 寵物詳情介面
- 大圖顯示區域
- 基礎數值：HP、攻撃、回復
- 隊長技能說明
- 主動技能說明
- 覺醒技能列表
- 點擊右上角 X 關閉

## 安裝與執行

### 需求
- Python 3.8+
- Pygame 2.0+

### 安裝依賴
```bash
pip install -r requirements.txt
```

### 執行遊戲
```bash
python main.py
```

## 專案結構

```
pad-game/
├── main.py              # 主程式入口
├── config.py            # 設定檔
├── requirements.txt     # Python 依賴
├── data/
│   ├── pets.json        # 寵物資料庫 (JSON)
│   └── pets.py          # 寵物類別
└── ui/
    ├── fonts.py         # 中文字體載入
    ├── tab_bar.py       # 分頁系統
    ├── inventory.py     # 背包系統
    ├── codex.py         # 圖鑑系統
    └── pet_detail.py    # 寵物詳情介面
```

## 遊戲操作

| 操作 | 功能 |
|------|------|
| 點擊分頁 | 切換背包/圖鑑 |
| 點擊寵物 | 開啟寵物詳情 |
| 拖曳寵物 | 移動寵物位置 |
| 點擊 X | 關閉詳情介面 |

## 新增寵物方法

打開 `data/pets.json`，在 `pets` 陣列中添加新物件：

```json
{
  "id": 17,
  "name": "新寵物名",
  "attribute": "火",
  "race": "神",
  "rarity": 6,
  "hp": 3000,
  "attack": 2000,
  "recovery": 300,
  "leader_skill": "隊長技能描述",
  "active_skill": "主動技能描述",
  "awakenings": ["覺醒1", "覺醒2"],
  "description": "寵物描述"
}
```

## 當前寵物

遊戲包含 16 種寵物：
- 5屬性埃（神族）
- 火/水/木龍（龍族）
- 金屬龍（龍族）
- 白虎/黑狗（獸族）
- 機械天使（機械族）
- 惡魔少女（魔族）
- 妖精女王（妖族）
- 劍豪/巫女（人族）
